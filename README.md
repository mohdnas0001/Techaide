# Techaide — Premium Electronics & Accessories

> Futuristic Next.js landing page for **Techaide**, Abuja's trusted destination for premium smartphones, laptops, electronics, and accessories.

## ✨ Features

- ⚡ Futuristic dark/tech aesthetic with cyan glow effects
- 📱 Fully responsive (mobile-first design)
- 🔳 Live QR Code with PNG download
- 🎬 Framer Motion scroll-triggered animations
- 🎨 Glassmorphism cards with animated grid background
- 🔗 Instagram & TikTok social links

## 🛠 Local Development

```bash
# Clone the repository
git clone https://github.com/mohdnas0001/Techaide.git
cd Techaide

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## ☁️ Deploy to Vercel

### One-click deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/mohdnas0001/Techaide)

### Manual steps

1. Go to [vercel.com](https://vercel.com) and sign in
2. Click **"Add New Project"**
3. Import `mohdnas0001/Techaide`
4. Click **Deploy** — done! 🎉
